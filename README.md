# RLCO_IFPA_ANANINDEUA
RLC Otimizador – Algoritmo de Metaheurísticas fundamentadas na análise Eletromagnética de Circuitos Elétricos. Estratégia de otimização estocástica inspirada na fenomenologia Física de circuitos elétricos em corrente alternada. O RLCO é uma ferramenta eficaz e fisicamente interpretável para problemas complexos das Ciências e das Engenharias.
